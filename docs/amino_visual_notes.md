# Amino Apps - Análise Visual Detalhada

## Paleta de Cores
- Background principal: #1a1a2e (azul escuro profundo)
- Background cards: #16213e (azul escuro mais claro)
- Background header/nav: #0f0f1a (quase preto com tom azul)
- Cor de destaque primária: #2dbe60 (verde Amino - botões CHECK IN)
- Cor de destaque secundária: #5b5fc7 (roxo/lilás - ícones)
- Texto principal: #ffffff (branco)
- Texto secundário: #8e8ea0 (cinza claro)
- Borda/outline: #2a2a4a (azul escuro sutil)
- Badge/notificação: #ff4444 (vermelho)
- Coins badge: #ffa500 (laranja/amarelo)

## Bottom Navigation Bar (4 tabs)
1. **Discover** - ícone de bússola/caneta
2. **Communities** - ícone de grid (4 quadrados)
3. **Chats** - ícone de balão de chat
4. **Store** - ícone de loja/prédio

## Top Bar (Header)
- Avatar do usuário (esquerda, circular)
- Barra de busca "Search" (centro)
- Seletor de idioma "EN" (dropdown)
- Badge de coins (verde com número, ícone de moeda)
- Botão "+" (verde, criar)
- Ícone de sino/notificação (direita)

## Tela Discover (Home)
- Banner grande no topo (carousel com imagens de comunidades)
- Dots de paginação abaixo do banner
- Seção "My Communities" com cards horizontais
- Cada card: imagem de fundo + nome da comunidade + botão "CHECK IN" verde
- Seção "Recommended Communities" com scroll horizontal
- Cards menores com thumbnails

## Tela Communities
- Título "My Communities"
- Cards de comunidades em grid horizontal
- Cada card: imagem, nome, botão "CHECK IN" verde
- Botão "+" para adicionar comunidade
- Texto "Long press the card to change position"
- Botão grande "CREATE YOUR OWN" com borda verde

## Tela Chats
- Sidebar esquerda com ícones de comunidades (vertical)
  - Recent (ícone relógio)
  - Global (ícone globo com badge)
  - Ícones das comunidades que participa
  - Botão "+" para adicionar
- Área principal:
  - "Create a new Chat" botão com "+"
  - Lista de chats com: avatar, nome, preview da mensagem, tempo
  - Seção "Recommended" com cards de chat público (imagem + nome + membros)

## Tela Me (Perfil)
- Não visível nas screenshots, mas da engenharia reversa:
  - Avatar, nickname, nível
  - Bio, seguidores, seguindo
  - Posts, reputação
  - Badges e conquistas

## Dentro da Comunidade
- Sidebar lateral com navegação
- Feed com posts
- Chat rooms
- Wiki/catálogo
- Membros e ranking

## Estilo Visual Geral
- Design MUITO escuro (dark navy blue)
- Cantos arredondados nos cards
- Sem glassmorphism - é mais flat/material
- Ícones outline brancos
- Botões verdes sólidos (CHECK IN, CREATE)
- Badges coloridos para notificações
- Layout denso com muita informação
- Sidebar de comunidades no chat (ícones verticais)
